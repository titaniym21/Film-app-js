strat page

- після пошити перехід на сторінку із реєстрацією
- додати плавні переходи між сторінками
- додати брейкпоінти під різні розміри екранів (адаптивність сторінки)
- доробити дизайн сторінки


- якщо в localStorage є дані, про користувача то відобразити іконку користувача 
- забрати кнопку вводу пошти


login page

- додати форму перевірки вводу пошти і паролю та зберігання даних в localStorage
- додати кнопку відправки форми
- додати кнопку реєстрації
- додати кнопку відновлення паролю, яка буде відображати данні з localStorage
- додати до іконки авторизованого користувача випадаючий список з  пунктами: 
  - профіль
  - вихід
- додати перехід в налаштування профілю


maim page

- додати слайдер для перелистування іконок категорій
  - додати можливість завантаження в слайдер інших сторінок окрім першої
- при наведенні на картинку фільму повинна збільшитись картинка
- при натисканні на картинку повинна збільшуватись майже на весь єкран та відображатись інформація про фільм
    - додати кнопки перегляду трейлеру, додавання в список перегляду, додавання в список бажань, закриття вікна
    - анімація при відкритті вікна з інформацією про фільм
- додати можливість пошуку фільму по назві
    - відображення через випадаючий список
- можливо додати API ютуба для перегляду трейлерів
